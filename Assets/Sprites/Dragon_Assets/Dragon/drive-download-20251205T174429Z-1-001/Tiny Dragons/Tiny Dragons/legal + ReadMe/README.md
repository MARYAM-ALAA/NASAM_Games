# Tiny Dragon Sprite Pack (64x64, 6-frame)

Chunky front-facing tiny dragons with a smooth 6-frame bounce. Each variant includes a 384x64 PNG spritesheet (six 64x64 frames) and a matching JSON atlas.

## Included Dragons
Fire, Ice, Shadow, Gold, Toxic, Crystal, Plushie, Rainbow

## Specs
- Frame: 64x64
- Frames: 6
- Sheet: 384x64 (single row)
- Loop: Bounce (smooth)
- BG: Transparent
- Files: PNG + JSON (same folder)

## How to Use
1. Import the PNG into your engine (Unity, Godot, Phaser, etc.).
2. Use the JSON to map frames in order and loop the "bounce" animation.
3. Keep X static for hover; move X yourself for travel.
